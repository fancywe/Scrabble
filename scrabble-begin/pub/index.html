How to play:


1. There are 100 blocks, each round a player will be given 7 blocks.


2. The system will check the words whether it in dictionary, the order is from top to button and from left to right.


3. If player can't make any words, player can click "pass" button to skip this round.


4. Each letter have their score, put letter on space which have "TL","DL","TW","DW" can get addition score.


5. Player who use all blocks will win, if both of them use all blocks, who has higher score will win.
